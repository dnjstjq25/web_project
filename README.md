## 제작도구
**사용 프로그램**
* Sublime Text3 -> HTML, CSS, JavaScript 작성
* Pixlr Editor -> 홈페이지에 들어가는 이미지 관리를 위해 사용

**사용 언어**
* HTML -> 화면 기본 틀 구성
* CSS -> 디자인
* JavaScript -> 간단한 이벤트 처리
